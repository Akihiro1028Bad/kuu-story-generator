# ステップインジケーター（①②③）デザイン改善案

## 📋 現在の実装分析

**現在のステップインジケーターの特徴**:
- プログレスバーと円形のステップ番号（1, 2, 3）
- グラデーションカラー（紫→ピンク→シアン）
- バウンスアニメーション
- 完了時にチェックマーク表示
- 波打つプログレスバーアニメーション

**改善が必要な点**:
- 視認性が悪い（数字が小さく、コントラストが低い可能性）
- モダンなデザイントレンドの反映
- 遊び心のあるマイクロインタラクションの追加
- プロフェッショナルな見た目の向上

---

## 🎨 デザイン案（全6案）

### 案1: ✨ グラデーション・フロー（Gradient Flow）

**コンセプト**: 流れるようなグラデーションとスムーズなアニメーション

**デザイン特徴**:
- 🌊 **流れるプログレスバー**: グラデーションが左から右へ流れるアニメーション
- 💫 **パルス効果**: アクティブなステップにパルスアニメーション
- 🎯 **3D浮き上がり**: アクティブステップが浮き上がる（elevation効果）
- ✨ **光るエッジ**: プログレスバーに光が走る（shimmer effect）
- 🎨 **グラデーションカラー**: 紫→ピンク→シアンの流れるグラデーション

**ステップ番号のデザイン**:
- 大きめの円形（直径: 48px）
- 太字の数字（font-weight: 700, font-size: 18px）
- 白いテキストで高いコントラスト
- グラデーション背景にドロップシャドウ
- アクティブ時はスケールアップ（1.15倍）

**アニメーション詳細**:
```css
/* プログレスバーの流れるグラデーション */
.progress-bar {
  background: linear-gradient(
    90deg,
    #8b5cf6 0%,
    #ec4899 50%,
    #06b6d4 100%
  );
  background-size: 200% 100%;
  animation: flowGradient 3s ease infinite;
}

/* ステップ番号のパルス */
@keyframes pulse {
  0%, 100% { 
    transform: scale(1); 
    box-shadow: 0 0 0 0 rgba(236, 72, 153, 0.7); 
  }
  50% { 
    transform: scale(1.1); 
    box-shadow: 0 0 0 10px rgba(236, 72, 153, 0); 
  }
}

/* 光るエッジ */
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
```

**遊び心要素**:
- 🎆 ステップ完了時に小さなパーティクルが飛び散る
- 🌈 ホバー時に虹色のグローが走る
- ✨ アクティブステップにキラキラエフェクト

**視認性の改善**:
- 数字のサイズを18pxに拡大
- フォントウェイトを700（太字）に設定
- 白いテキストでコントラスト比を確保（WCAG AA準拠）
- ドロップシャドウで数字を強調

### 案2: 🎪 バウンス・カーニバル（Bounce Carnival）

**コンセプト**: 楽しくエネルギッシュなバウンスアニメーション

**デザイン特徴**:
- 🎈 **バウンスアニメーション**: ステップが跳ねるような動き
- 🎨 **カラフルなカラー**: 各ステップに異なる鮮やかな色
- 💫 **回転エフェクト**: アクティブ時にステップが回転
- 🎯 **3D効果**: ステップが立体的に見える
- ✨ **スパークル**: アクティブステップにキラキラエフェクト

**ステップ番号のデザイン**:
- 大きめの円形（直径: 52px）
- 太字の数字（font-weight: 800, font-size: 20px）
- 白いテキストで高いコントラスト
- 各ステップに異なるグラデーション
  - ステップ1: 紫→ピンク
  - ステップ2: ピンク→オレンジ
  - ステップ3: オレンジ→イエロー
- 立体的なシャドウ（multiple box-shadow）

**アニメーション詳細**:
```css
/* バウンスアニメーション */
@keyframes bounce {
  0%, 20%, 50%, 80%, 100% {
    transform: translateY(0) scale(1);
  }
  40% {
    transform: translateY(-12px) scale(1.1);
  }
  60% {
    transform: translateY(-6px) scale(1.05);
  }
}

/* 回転エフェクト */
@keyframes rotate {
  0% { transform: rotate(0deg) scale(1.1); }
  100% { transform: rotate(360deg) scale(1.1); }
}

/* 3D効果 */
.step-number {
  transform-style: preserve-3d;
  perspective: 1000px;
}
```

**遊び心要素**:
- 🎉 ステップ完了時に花火のようなエフェクト
- 🎈 ホバー時にバルーンが浮かび上がる
- 🎪 アクティブ時にカーニバルのような装飾

**視認性の改善**:
- 数字のサイズを20pxに拡大
- フォントウェイトを800（超太字）に設定
- 各ステップに異なる色で識別しやすく
- 立体的なシャドウで数字を強調
